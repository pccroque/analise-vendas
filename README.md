# Análise de Vendas de uma Loja Virtual
Projeto de análise de dados de vendas usando Python, Pandas, Matplotlib e Seaborn em Jupyter Notebook. O estudo examina dados fictícios de uma loja virtual para identificar padrões, entender o comportamento dos clientes e gerar insights que apoiem decisões estratégicas.

# Objetivo:
Analisar os dados de vendas de uma loja virtual para identificar os produtos mais vendidos, sazonalidade, ticket médio e comportamento dos clientes.

# Ferramentas utilizadas:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

# Resumo das etapas realizadas:

1. Importação e limpeza dos dados:
- Remoção de duplicatas
- Tratamento de valores nulos
- Conversão de datas e tipos numéricos
  
2. Análise exploratória:
- Produtos mais vendidos
- Dias da semana com mais vendas
- Ticket médio por cliente
- Gráficos de linha, barras e pizza

3. Principais insights:
- Segunda-feira é o dia com maior volume de vendas
- O produto “Camisa Polo” representa 18% do faturamento total
- Clientes recorrentes gastam em média 35% a mais que novos clientes

4. Entrega final:
- Relatório em PDF com gráficos e insights
- Arquivo Excel com dados limpos
- Código comentado em Jupyter Notebook
