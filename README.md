# Análise de Vendas de uma Loja Virtual
Projeto de análise de dados de vendas usando Python, Pandas, Matplotlib e Seaborn em Jupyter Notebook. O estudo examina dados fictícios de uma loja virtual para identificar padrões, entender o comportamento dos clientes e gerar insights que apoiem decisões estratégicas.

# Objetivo:
Analisar os dados de vendas de uma loja virtual para identificar os produtos mais vendidos, sazonalidade, ticket médio e comportamento dos clientes.

# Ferramentas utilizadas:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

# Resumo das etapas realizadas:

## 1. Importação dos Dados de uma Planilha Excel
Utilizaremos no projeto um dataset fictício que simula vendas de uma loja virtual, contendo informações como data da venda, 
produto, quantidade, preço unitário, cliente e categoria do produto.

Arquivo: **dados_vendas_ficticios.xlsx**

## 2. Limpeza dos Dados
- Remoção de duplicatas.
- Tratamento de valores nulos.
- Conversão de datas e tipos numéricos.

## 3. Análise Exploratória dos Dados
- Produtos mais vendidos
- Dias da semana com mais vendas
- Ticket médio por cliente
- Visualizações gráficas com Matplotlib e Seaborn

## 4. Principais insights

#### Após a análise, destacamos os principais insights e entregamos os resultados.
- A "Camisa Polo" é o produto mais vendido, seguido pelo "Boné".
- Quarta-feira é o dia com maior volume de vendas.
- O cliente_34 é o cliente de alto valor.
- "Camisa Polo" representa uma fatia significativa do faturamento.
- Clientes recorrentes gastam mais que novos clientes.

- Relatório em PDF com gráficos e insights

- Código comentado em Jupyter Notebook
